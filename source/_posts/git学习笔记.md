##### 背景

主流的版本控制器

- git
- svn
- ...

版本控制分类

- 本地版本控制
- 集中版本控制（svn）
- 分布式版本控制（git）

##### git核心原理

git本地有三个工作区域：

- 工作目录：本地存放代码的位置
- 暂存区：本质上是一个文件，临时存放改动
- 资源库：仓库区
- 远程仓库：GitHub，gitee等

##### git核心命令

- add
- commit
- push
- pull
- checkout
- status
- fetch
- remote
